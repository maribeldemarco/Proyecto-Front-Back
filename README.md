
# **.env**

Crear este archivo para las variables de entorno y colocarlo junto al archivo **index.js**.  
Este tiene actualmente las siguientes variables:  
>
> EXPRESS_PORT = valor  
> EXPRESS_HOST = valor
> 
(Se ira actualizando según se creen más variables)
